What This App Is

GutWell is an educational demonstration project exploring how AI can be used to build practical, real-world health tools. It was created to showcase prompt engineering and AI-assisted app development using Claude by Anthropic.

What It Offers

A structured gut health protocol with Caribbean-specific meal guidance, herb rotation, daily symptom tracking, journaling, and a wellness reference guide — all for general educational purposes.

Important Disclaimer

This app is for educational and demonstration purposes only. It is not a medical application and does not provide medical advice. Always consult a qualified healthcare practitioner before making any changes to your diet, supplements, or health routine.

I understand this is an educational demonstration project and not a substitute for professional medical advice.
